# This app is Created by using
> HTML 5
> TailwindCSS
> Postcss
> React.js
> Chart.js
> Playwright for Tests

## to run it 
 npm install
 npm start

## Starting point is
 src/index.js ---> src/App.js ---> then components....etc

 ## for writing tests
npx playwright codegen http://localhost:3000/
 ## for running tests
npx playwright test example.spec.js --headed
